Name Day Backend API Documentation

Welcome to the Name Day Backend API documentation. This RESTful API provides international name-day data for Poland and Hungary. Name days are traditionally celebrated in many European cultures and are based on historical calendars and saints.

1. Base URL

https://name-day-backend-0d74dcea0ed2.herokuapp.com/

2. API Endpoints

Get All Name Days

  • Endpoint: /
  • Method: GET
  • Description: Retrieve name-day data for all months and supported countries.
  • Example Request:
    https://name-day-backend-0d74dcea0ed2.herokuapp.com/

Get Name Days by Month

  • Endpoint: /:month
  • Method: GET
  • Description: Retrieve name-day data for a specific month.
  • Parameter: month â€” lowercase month name (e.g. january)
  • Example Request:
    https://name-day-backend-0d74dcea0ed2.herokuapp.com/january

Get Name Days by Month and Date

  • Endpoint: /:month/:date
  • Method: GET
  • Description: Retrieve name-day data for a specific date.
  • Parameters: month, date
  • Example Request:
    https://name-day-backend-0d74dcea0ed2.herokuapp.com/january/1

Get Name Days by Month, Date, and Country

  • Endpoint: /:month/:date/:country
  • Method: GET
  • Description: Retrieve name-day data for a specific country.
  • Supported countries: hu, pl
  • Example Request:
    https://name-day-backend-0d74dcea0ed2.herokuapp.com/january/1/hu

3. Error Handling

If data cannot be found for the requested parameters, the API returns a 404 Not Found response.

{
  "error": "Data not found"
}
