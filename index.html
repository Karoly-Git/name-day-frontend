API Endpoints

Get All Name Days

  • Endpoint: /api/v1/namedays
  • Method: GET
  • Description: Retrieve all name-day data.
  • Example:
    /api/v1/namedays

Get Name Days by Month

  • Endpoint: /api/v1/namedays/:month
  • Method: GET
  • Parameter: month — lowercase month name
  • Example:
    /api/v1/namedays/january

Get Name Days by Month and Date

  • Endpoint: /api/v1/namedays/:month/:date
  • Method: GET
  • Parameters: month, date
  • Example:
    /api/v1/namedays/january/1

Get Name Days by Month, Date, and Country

  • Endpoint: /api/v1/namedays/:month/:date/:country
  • Method: GET
  • Supported countries: pl, hu
  • Example:
    /api/v1/namedays/january/1/pl

The API supports the following routes:

  • GET /api/v1/namedays/api-docs — OpenAPI documentation (Swagger UI)

  • GET /api/v1/namedays/:month — month data
  • GET /api/v1/namedays/:month/:date — specific day data
  • GET /api/v1/namedays/:month/:date/:country — specific country data
