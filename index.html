2. About the Service

The Name Day API provides name day data for supported countries (currently Hungary and Poland). Name days are cultural observances and may vary by source.

3. Base Path and Endpoints

The Service is available under the following base path:

https://nameday-b9bd663a9ed7.herokuapp.com/api/v1/namedays

The API supports the following routes:

  • GET /api/v1/namedays/api-docs — OpenAPI documentation (Swagger UI)
  • GET /api/v1/namedays — all data
  • GET /api/v1/namedays/:month — month data
  • GET /api/v1/namedays/:month/:date — specific day data
  • GET /api/v1/namedays/:month/:date/:country — specific country data
